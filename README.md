Fault-Tolerant Self-Checking Framework for Vision-Language Models

A lightweight self-checking framework for detecting semantic inconsistencies in image captions generated by Vision-Language Models (VLMs).

This project explores whether semantic similarity can serve as a fault detection mechanism for image captioning systems using BLIP and Sentence-BERT.

Modern Vision-Language Models can generate fluent image descriptions, but they may also produce:

hallucinated objects
semantic inconsistencies
incorrect visual descriptions

This project implements a lightweight fault-tolerant framework that:

Generates captions using BLIP
Injects synthetic semantic faults
Uses Sentence-BERT cosine similarity to detect inconsistencies

The system evaluates whether semantic similarity can identify faulty captions in a computationally efficient manner.
